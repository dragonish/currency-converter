# 多币种汇率转换工具

<!-- markdownlint-disable no-inline-html -->
<img alt="icon" width="68" src="images/icon.png" style="display: block; margin: auto;" />

这是一款支持多币种实时兑换的网页工具，界面美观，交互友好，支持深色/浅色/跟随系统三种主题模式。汇率数据来源于多个权威 API，并在网络不可用时提供本地模拟数据，保证随时可用。

> [源代码](https://github.com/kejilion/Website_source_code/blob/main/currency-converter/index.html)

## 核心功能

### 实时汇率获取

多数据源（ExchangeRate-API、Frankfurter）自动切换，确保稳定性。

自动更新时间，支持一键刷新。

### 多币种支持

默认支持 20+ 主流货币（美元、欧元、人民币、日元、英镑、港币、新台币等）。

每种货币均配有国旗和中文名称，直观易识别。

### 双向兑换计算

任意输入一个币种金额，即可实时换算成其他货币。

自动设定输入币种为基准货币，所有汇率即时更新。

### 可拖拽排序

内置拖拽功能，支持用户自定义币种顺序。

排序结果自动保存，下次打开页面仍保持个性化配置。

### 历史趋势分析

点击任意货币即可查看 历史汇率走势图。

支持 5天 / 30天 / 1年 / 5年 多时间范围切换。

展示关键统计数据：起始汇率、结束汇率、涨跌幅、最高点、最低点。

### 个性化主题

浅色（蓝白风格）、深色（金黑风格）、跟随系统模式自由切换。

自动适配系统主题变化。
